我在这里，就不展示我们以前的组件实现了。我们只实现了React-loadable最基础的功能，事实上它还能够有以下几点优势：

  • 当加载组件的过程中，给出过渡效果
  • 当加载失败、加载错误后，给出一个类似404的页面，不致于白屏
  • 能够进行retry 等行为
